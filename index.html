⚓ Argazer
Monitor ArgoCD applications for Helm chart updates and get notified

Features

🔍
ArgoCD Integration
Connects via gRPC-Web API, filters apps by project, name, and labels
📦
All Repo Types
HTTP Helm repos, OCI registries, and Git-based charts — all supported
🔔
Multi-Channel Notifications
Telegram, Email, Slack, Microsoft Teams, or generic Webhook
✏️
Custom Message Templates
Define your own Go text/template for notification messages
🔒
Private Registries
Authenticate to multiple private Helm repos and OCI registries
⚙️
Flexible Config
config.yaml, environment variables, or CLI flags — all supported

Install via Helm

1
Add the Helm repository
helm repo add vorkerj https://vorkerj.github.io/argazer
helm repo update
2
Create a Kubernetes Secret with your credentials
kubectl create secret generic argazer-credentials \
  --from-literal=argocd-username=admin \
  --from-literal=argocd-password=your-password \
  --from-literal=telegram-webhook=https://api.telegram.org/bot.../sendMessage \
  --from-literal=telegram-chat-id=123456789
3
Install the chart
helm install argazer vorkerj/argazer \
  --set schedule="0 */6 * * *" \
  --set argocd.url="argocd.example.com" \
  --set argocd.existingSecret="argazer-credentials" \
  --set notifications.channel="telegram" \
  --set notifications.telegram.existingSecret="argazer-credentials"
Or use a values.yaml file: helm install argazer vorkerj/argazer -f my-values.yaml

Custom Message Template

Define your own notification format using Go text/template syntax. Falls back to the default format on any error. 

# values.yaml
messageTemplate: |
  🚀 *{{.AppName}}* ({{.Project}})
  📦 {{.ChartName}}: {{.CurrentVersion}} -> {{.LatestVersion}}
  🔗 {{.RepoURL}}
FieldExample
.AppNamecert-manager
.Projectinternal
.ChartNamecert-manager
.CurrentVersion1.18.5
.LatestVersionv1.20.0
.RepoURLhttps://charts.jetstack.io
.ConstraintAppliedminor
.HasUpdateOutsideConstrainttrue
.LatestVersionAll2.0.0
